1. У инфоблока создано множественное свойство типа "Привязка к пользователю" с кодом USER_LIKES.
2. В корне сайта создан файл ajax.handler.php с вызовом компонента     
<? $APPLICATION->IncludeComponent("ajax:iblock.element.add.form","", array(), false);  ?>
3. В этом компоненте происходит проверка и изменение массива свойств USER_LIKES. 
4. Обновление полей осуществляется с помощью CIBlockElement::SetPropertyValuesEx.
5. В шаблоне компонента выводится список пользователей, которым статья нравится.
6. В скрипте отправляется ajax запроса к ajax.handler.php со следующими параметрами:
	а) идентификатор статьи
	б) идентификатор пользователя
	в) параметр-флаг (нравится или не нравится статья, т.е. добавлять или удалять запись из массива USER_LIKES)
